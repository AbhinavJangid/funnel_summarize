# Ticket Copilot Snap-in

The Support Copilot Snap-In for Ticket Objects unifies related issues, opportunities, and account information directly within your ticket interface. Gain comprehensive context and intelligent insights to resolve tickets more efficiently. Enhance your decision-making process and streamline ticket management with seamless data convergence.

# How to use the Snap-in

- After deploying the Snap-in, navigate to timeline of any ticket.
- Trigger the command using `/copilot`.
